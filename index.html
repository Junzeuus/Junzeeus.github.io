Pertanyaan Umum (FAQ)

Berapa lama proses top up?

Proses top up di GameTopUp sangat cepat, biasanya hanya memakan waktu 1-5 menit setelah pembayaran dikonfirmasi. Jika terjadi gangguan, maksimal proses adalah 1x24 jam.

Metode pembayaran apa saja yang tersedia?

Kami menyediakan berbagai metode pembayaran, mulai dari E-Wallet (Dana, OVO, GoPay, ShopeePay), QRIS, Transfer Bank (BCA, Mandiri, BRI, BNI), hingga pembayaran via Minimarket (Indomaret, Alfamart).

Apakah top up di sini aman?

Sangat aman! Kami menggunakan sistem enkripsi data terbaru dan bekerja sama dengan gateway pembayaran resmi. Selain itu, kami juga bergaransi uang kembali jika top up gagal.

Bagaimana jika saya salah memasukkan ID?

Mohon maaf, kesalahan memasukkan User ID sepenuhnya menjadi tanggung jawab pengguna. Pastikan Anda memeriksa kembali User ID sebelum melakukan pembayaran.
